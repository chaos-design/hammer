## 使用方式

```tsx
import { MonthDatepicker } from "@chaos-design/month-datepicker";
```

## 在 shadcn/ui 中使用

1. 安装依赖

```bash
npm install @chaos-design/month-datepicker
```

2. 在组件中使用

```tsx
import { MonthDatepicker } from "@chaos-design/month-datepicker";

export default function Demo() {
  return <MonthDatepicker />;
}
```
